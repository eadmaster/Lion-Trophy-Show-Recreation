
# The Lion Trophy Show Recreation

<img src="https://raw.githubusercontent.com/eadmaster/Lion-Trophy-Show-Recreation/refs/heads/main/assets/logo.png" alt="demo1" width="300">

Recreation of an [old FMV game](https://it.wikipedia.org/wiki/Lion_Network) broadcast on Italian TV in the 90s.

[Play online here](https://eadmaster.github.io/Lion-Trophy-Show-Recreation/), or [download the repository](https://github.com/eadmaster/Lion-Trophy-Show-Recreation/archive/refs/heads/main.zip) to play it offline.
